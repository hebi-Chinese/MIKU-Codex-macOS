# MIKU · 爱言叶公开预设

这是一套可随仓库分发、安装后可直接选择的 MIKU Codex 主题包。

## 文件

- `background.png`：主窗口背景，`1672 × 941`，SHA-256 `09147b7c58da8aa4e25e43fdb1ebe4a66024ed557e5c7b8ceda582b61b237d33`。
- `side-chat-background.png`：由同一背景右侧构图裁切出的侧聊专用图，`1200 × 1200`，SHA-256 `488380702789ae08c005097153baf025c534cb04763b64bacf32ec6391837298`。
- `theme.json`：公开、可移植的主题元数据；不包含维护者的本机项目名、路径或任务数据。
